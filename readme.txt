== Description ==

Widget to show bookmarks with exclude option, extra params are possible

== Installation ==

1. Upload `bookmarks_exclude.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit Appearance > Widgets and configure the plugin
